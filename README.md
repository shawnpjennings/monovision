# MONOvision™

**A Vandalway Industries company**  
**Owner:** Jorge Procanto  
**Codename:** Project Close One Eye  
**Status:** Pre-Beta / Post-Argument

MONOvision™ is an adaptive optical productivity system for people who looked at modern vision correction and thought, “What if both eyes were simply assigned different jobs and then refused to collaborate?”

It is not monovision. That would be cowardice. MONOvision is a proprietary eyelid-toggle platform that lets users switch between near and far focus by closing whichever eye is currently causing organizational friction.

## The Pitch

Modern optometry asks:

> “How can we help both eyes work together?”

MONOvision asks:

> “What if they didn’t?”

The result is **situational optical sovereignty**, a premium-sounding phrase that mostly means the eyelids were already there and nobody had the courage to productize them.

## Core Modes

| Mode | Active Strategy | Notes |
|---|---|---|
| `ReadMode` | Near eye active | Great for menus, receipts, and silently judging a subtotal. |
| `DistanceMode` | Far eye active | Useful for exits, snacks, and identifying whether someone is waving or stretching. |
| `SuspicionMode` | Narrowed-eye ambiguity | For emotionally complicated diners and condo bulletin boards. |
| `MenuPanicMode` | Rapid blink fallback | Activated when the specials are handwritten. |
| `ParkingGarageMode` | Unsupported | We are not animals. |

## Tech Stack

MONOvision runs on the Vandalway Almost-Reasonable Stack:

- **Close.js**: reactive UI framework that renders only when one viewport is unavailable.
- **PoofShirt CSS**: utility styling with questionable spacing opinions.
- **BrothDenied UI Kit**: components for booths, counters, menus, and interpersonal discomfort.
- **Thumper Runtime**: app server with enthusiastic but unreliable request handling.
- **FestivaxDB**: document database with built-in grievance indexing.
- **JorgeNet**: inference engine optimized for the worst possible interpretation of a normal decision.
- **YottaYotta Embeddings**: compresses long explanations into suspiciously missing context.
- **Oppositron Vector Store**: returns semantically opposite but emotionally relevant results.
- **SerenityNow Queue**: background jobs, yelling optional but encouraged.
- **DelBocaVista Edge Nodes**: regional compute deployed near pool furniture.
